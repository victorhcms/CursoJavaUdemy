***OPERADORES COMPARATIVOS***

Operador / Significado
>  / maior
<  / menor
>= / maior ou igual
<= / menor ou igual
== / igual
!= / diferente

**OPERADORES LÓGICOS**
Operador / Significado
&& / E
|| / OU
! / NÃO

**OPERADORES ARITMÉDICOS**
Operador / Significado
+ / adição
- / subtração
* / multiplicação
(/) / divisão
% / resto da divisão ("mod")
